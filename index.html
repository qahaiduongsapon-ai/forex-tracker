🔗 Kết nối Google Sheets

Lưu dữ liệu lên Google Sheets để dùng chung trên mọi thiết bị.

Bước 1: Mở Google Sheets → Extensions → Apps Script
Bước 2: Xoá code mặc định → Dán code từ file GoogleAppsScript_ForexTracker.js
Bước 3: Deploy → New deployment → Web app
    Execute as: Me | Who has access: Anyone
Bước 4: Copy Web app URL và dán vào ô bên dưới
